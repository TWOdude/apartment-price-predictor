# Apartment Price Predictor

Дипломный проект по машинному обучению для предсказания рыночной стоимости квартир в Санкт-Петербурге.

## 📋 Описание проекта

Проект состоит из нескольких этапов:
1. **Сбор данных** - парсинг объявлений с Авито
2. **Обработка данных** - очистка и подготовка признаков
3. **Обучение модели** - разработка ML моделей
4. **Оценка и улучшение** - оптимизация и валидация

## 🎯 Целевые признаки

- **Цена** (целевая переменная)
- Площадь (м²)
- Количество комнат
- Этаж
- Год постройки
- Район/адрес
- Расстояние до метро (км)

## 🔧 Установка и использование

### Требования
- Python 3.8+
- pip

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Запуск парсера
```bash
python -m scraper.avito_scraper
```

## ⚙️ Конфигурация

Параметры парсера настраиваются в `scraper/config.py`

## 📊 Данные

Собранные данные сохраняются в CSV формате в папке `data/raw/`
